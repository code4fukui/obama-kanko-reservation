## 機能
- 福井県小浜地域のホテルの予約状況を表示します
- 客室総数、稼働率、平均1泊料金、 RevPAR など、詳細なデータを提供します
- データを視覚的に表すチャートも含まれています
- データは福井県観光連盟から収集しています

## データ/API
このプロジェクトでは、福井県観光連盟が提供する「FTAS」観光データ分析システムのオープンデータを使用しています。
