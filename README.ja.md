# obama-kanko-reservation
[README in Japanese is here: README.ja.md](README.ja.md)

https://code4fukui.github.io/obama-kanko-reservation/

## 特徴
- 日本、福井県小浜市の宿泊施設の予約状況を表示します
- 総客室数、稼働率、平均宿泊料金、1室当たり収益など、詳細なデータを提供します
- データの視覚化チャートを含んでいます
- データソースは福井県観光連盟です

## データ / API
このプロジェクトは、福井県観光連盟が提供する「FTAS」観光データ分析システムからのオープンデータを使用しています。
